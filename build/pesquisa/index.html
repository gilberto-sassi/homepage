Pacotes no R

Pacotes desenvolvidos e mantidos por mim.

geoFKF

Este pacote é uma implementação da minha tese de doutorado, onde eu propus um novo método de interpolação de curvas com dependência espacial dentro do contexto da Análise Funcional de Dados. Para mais detalhes, sobre o método proposto consulte minha tese de doutorado em Banco de teses da USP. Em minha tese de doutorado, usei ondaletas, mas neste pacote usei Séries de Fourier por sua simplicidade e apelo prático.

Este pacote está disponível no repositório CRAN em geoFKF. Para maiores detalhes, configura o repositório do pacote no GitHub em gilberto-sassi/geoFKF.

geoFourierFDA

Este é uma implementação do método proposto em

GIRALDO, R.; DELICADO, Pedro; MATEU, Jorge. Ordinary kriging for function-valued spatial data. Environmental and Ecological Statistics, v. 18, n. 3, p. 411-426, 2011.

Neste pacote, também usamos Séries de Fourier e o pacote está disponível para download no repositório CRAN em geoFourierFDA. Para maiores detalhes, configura o repositório do pacote no GitHub em gilberto-sassi/geoFourierFDA.

sinar

Este pacote é resultado de um projeto de Pesquisa, onde eu e a profa Carolina Costa Mota Paraíba propomos o uso do Método do Mínimos Quadrados para o modelo de contagem com dependência espacial denominado Spacial Integer Autoregressive Model (SINAR(1,1)). Neste pacote, incluímos a estimações dos parâmetros do modelo e o seus respectivos intervalos de confiança assintóticos.

Você baixar este pacote diretamente do repositório CRAN em sinar. Para maiores detalhes, configura o repositório do pacote no GitHub em gilberto-sassi/sinar e o seguinte artigo:

PEREIRA SASSI, Gilberto; COSTA MOTA PARAÍBA, Carolina. Conditional least squares estimation for the SINAR (1, 1) process. Communications in Statistics-Simulation and Computation, p. 1-30, 2021.

statBasics

Criei este pacote para auxiliar no ensino dos cursos introdutórios de Estatística para alunos que não são do Bacharelado em Estatística, e ele está disponível no repositório CRAN em statBasics.

Para mais detalhes, confira o repositório no GitHub em statBasics.
